ADORe API Reference

POST /api/scenario/start
Start a scenario using either a file path or launch file content

Request Body:

{ "scenario": "path/to/scenario.launch.py" | "launch file content", "is_file": true | false }
POST /api/scenario/stop
Stop the currently running scenario
POST /api/scenario/restart
Restart the current scenario (halts all processes first, then restarts)
POST /api/scenario/halt
Halt all running scenarios and ROS2 processes
GET /api/scenario/output?lines=1000
Get stdout/stderr output from the running scenario

Query Parameters:

lines: number of lines to return (default: 1000)
GET /api/scenario/status
Get current scenario status and details

Response:

{ "status": "running" | "idle" | "failed", "scenario": "current_scenario_name", "scenario_content": "launch file content", "loop_mode": true | false, "loop_delay": 0, "default_runtime": 60, "runtime": 45.2, "pid": 12345 }
GET /api/scenario/get
Get list of all available scenarios

Response:

{"scenarios": ["scenario1.launch.py", "scenario2.launch.py"]}
GET /api/scenario/content/<scenario_path>
Get the content of a specific scenario file

Response:

{ "success": true, "content": "launch file content", "path": "scenario_path" }
POST /api/scenario/save
Save a custom scenario to the scenario directory

Request Body:

{ "name": "scenario_name", "content": "launch file content" }
POST /api/scenario/loop
Enable/disable loop mode with configurable runtime and delay

Request Body:

{ "enabled": true | false, "delay": 0, "runtime": 60 }
GET /api/positions/get
Get stored start and goal positions from the goal picker

Response:

{ "start": {"lat": 52.5200, "lng": 13.4050, "utm": {...}}, "goal": {"lat": 52.5300, "lng": 13.4150, "utm": {...}} }
POST /api/positions/set
Store start and goal positions from the goal picker

Request Body:

{ "start": {"lat": 52.5200, "lng": 13.4050, "utm": {...}}, "goal": {"lat": 52.5300, "lng": 13.4150, "utm": {...}} }
POST /api/positions/clear
Clear stored start and goal positions
