# ADORe Node Status

## Overview

**ADORe Node Status** is a tool for transmitting arbitrary standard types across arbitrary nodes using a shared msg and cpp type.

---

## Features
- **Arbitrary and safe standard type compresion to json** 
- **Arbitrary and safe standard type extration from json**
- **Quick overview text fields for easy lichtblick visualization and debugging**
- **Easy convertion between cpp and ros2 msg type**
- **Time of publication**
